# ECommerce

Este projeto foi gerado usando [Angular CLI](https://github.com/angular/angular-cli) versão 22.0.4.

## Servidor de Desenvolvimento
para iniciar um servidor de desenvolvimento locar, execute:

```bash
ng serve
```

Assim que o servidor estiver rodando, abra seu navegador e navege ate `http://localhost:4200/`. A aplicação será carregada automaticamente sempre que você modificar qualquer um dos arquivos fontes.

## Geração de código

O Angular CLI inclui ferramentas poderosas de geração de código. Para gerar um novo componente, execute:

```bash
ng generate component component-name
```

Para uma lista de esquemas disponíveis (como 'components', 'directives' ou 'pipes'), execute:

```bash
ng generate --help
```
## Compilação

Para compilar o projeto, execute:

```bash
ng build
```

Isso compilará seu projeto e armazenará os artefatos de compilação no diretório dist/. Por padrão, a compilação de produção otimiza sua aplicação para desempenho e velocidade .

## Executando testes unitários

Para executar testes unitários com o executor de testes ⁠Vitest, use o seguinte comando:

```bash
ng test
```

## Executando testes de ponta a ponta

Para testes de ponta a ponta (e2e), execute:

```bash
ng e2e
```

O Angular CLI não inclui, por padrão, um framework de testes de ponta a ponta (end-to-end). Você pode escolher aquele que melhor atenda às suas necessidades.
